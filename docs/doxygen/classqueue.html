Detailed Description

template<typename T, uint16_t N>
class queue< T, N >

Simple queue based on a stack-allocated circular buffer.

Template Parameters
Ttype of item this queue will store
Nmaximum number of items this queue can hold
Note
Internally, this queue allocates space for N+1 items on the stack to simplify size calculations
